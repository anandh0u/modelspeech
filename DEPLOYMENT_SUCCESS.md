**Project Name:** EmoMemory

**Tagline:** Memory-Enabled Emotion AI - AI That Never Forgets

**Description:**
Traditional AI systems suffer from amnesia - they forget everything after each interaction. EmoMemory uses Cognee's hybrid graph-vector memory system to create emotion AI that remembers across sessions, learns from patterns, and provides GDPR-compliant data management.

The demo shows all 4 core Cognee operations:
- REMEMBER: Store emotional states with context
- RECALL: Retrieve relevant past interactions  
- IMPROVE: Build knowledge from emotional patterns
- FORGET: Remove data on request (GDPR compliant)

The production version (on GitHub) includes full Cognee Cloud integration, advanced ML models, user authentication, wallet system, REST API, and Docker deployment.
