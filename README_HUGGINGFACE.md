# 🧠 EmoMemory - AI That Never Forgets

**Memory-Enabled Emotion Intelligence powered by Cognee**

Built for WeMakeDevs x Cognee Hackathon 2025

---

## 🎯 The Problem

Traditional AI systems suffer from **amnesia**:
- Every request is stateless
- No memory of past conversations
- Context window limits (tokens run out)
- Can't learn from user patterns

**It's like having a therapist who forgets everything you said!**

---

## 💡 The Solution

**EmoMemory** uses [Cognee](https://github.com/topoteretes/cognee) to give emotion AI a **permanent memory layer**:

✅ **REMEMBER** - Store emotional interactions persistently  
✅ **RECALL** - Retrieve relevant past contexts  
✅ **IMPROVE** - Build knowledge graph connections  
✅ **FORGET** - Remove data when needed (GDPR compliant)

---

## 🎬 Try It!

1. **First message:** "I'm excited about my new project!"
   - AI detects emotion
   - Stores in memory

2. **Second message:** "I'm worried about the deadline..."
   - AI detects emotion
   - **Recalls your project excitement from before!**

3. **Third message:** "I think we can make it work!"
   - AI understands the full emotional journey

This is the **magic of Cognee memory**! 🪄
