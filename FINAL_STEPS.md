## 🆘 Troubleshooting

### Problem: HF authentication failed
**Solution:** 
- Get token from: https://huggingface.co/settings/tokens
- Must have "Write" access
- Use token as password, not your HF password

### Problem: HF build failed
**Solution:**
- Check Logs tab in HF Spaces
- Usually fixes itself on retry
- If not, check requirements.txt

### Problem: Demo not working
**Solution:**
- Give it 2-3 minutes after push
- Click "Restart Space" if needed
- Check Logs for errors
